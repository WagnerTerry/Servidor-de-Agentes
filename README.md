# Servidor de agentes
- API para cadastro de agentes, desafio backend digitro

## Dependencias
- npm i typescript @types/node -D
- npm i tsx -D
- npm i express -D
- npm i --save-dev @types/express
- npm i cors
- npm i uuid
- npm i validator
- npm i mongoose

## Rodando o banco no docker
- docker exec -it backend_mongodb_1 mongosh -u root -p root
